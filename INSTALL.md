# 小钱本使用方式

## 电脑

1. 双击 `start-xiaoqianben-app.bat`。
2. 打开后就可以直接记账。
3. 资料会先保存在这台电脑的浏览器里面。

## iPhone

1. 先把小钱本上线到公开网址。
2. 用 Safari 打开小钱本网址。
3. 点分享按钮。
4. 选择「加入主屏幕」。
5. 以后就可以像应用一样从 iPhone 主屏幕打开。

## Android

1. 先把小钱本上线到公开网址。
2. 用手机浏览器打开小钱本网址。
3. 点「安装应用」或菜单里的「添加到主屏幕」。
4. 以后就可以像应用一样从手机主屏幕打开。

## 重要提醒

现在小钱本已经准备好给手机安装。  
手机要真的使用，需要先上线到一个网址；本机网址 `127.0.0.1` 只能在这台电脑自己打开。
